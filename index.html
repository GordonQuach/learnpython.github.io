True or False?
-Booleans can be set to true or false. You would use comparators to set comparators to given values.
Comparator True False
# Equal to (==) 5==5 5==6
# Not equal to (!=) (20/20) != 2 (30**5) != 24300000
# Less than (<) 5 < 6 80*30 < 5
# Less than or equal to (<=) 80*308000 < (80*308000+80*50) 80*308000 < 5
# Greater than (>) (888**9 + 50) > 4 999999999 > 10000000000000
# Greater than or equal to (>=) 7**2 >= 49 50 >= 100
-Here's an example of comparators within a method! 

						def TestingComparators():
							print ((20/20) != 2) #True
							print (7**2 >= 49) #True
							print (30 == 50) #False
							print (80*30 < 5) #False

						TestingComparators()
For Loops
-You set an element (such as variable n) after the for in a range.
-Now, the range be set to a max (101), exclusively (1,101) [which would only do 100 terms], and increments (1, 101, 2). This means that it will start at 1, end at 100, and increase by 2. Therefore, there would only be 50 counts.
-After you intialize the for loop, you can initialize a boolean, such as an if statement to run a counter. Every time the assertion is true within the boolean, the function will occur.
-Here's an example! 

						for n in range(101):
							if(n % 4 is 0):
							print(n)
